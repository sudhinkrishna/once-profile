3. Follow the format below exactly

## How proposals are evaluated

A proposed code is approved if it:
- Does not duplicate an existing code
- Is detectable from natural conversation without direct questions
- Applies to at least 2 different P parameters
- Is described in 15 words or fewer
- Adds meaningful signal not captured by combining existing codes

A proposed code is rejected if it:
- Duplicates or overlaps significantly with an existing code
- Requires self-reporting to detect
- Is too narrow to be reusable across profiles
- Is phrased as a value judgement rather than an observable behaviour
